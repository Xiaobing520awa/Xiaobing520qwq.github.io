# 程序员 Bug 修复手册

程序员鱼皮的编程宝典：https://codefather.cn/

# 介绍

程序员鱼皮带大家做项目多年，发现大家遇到的很多问题其实都是通用的。

为提高 Bug 的解决效率、带大家提升自主解决 Bug 的能力，鱼皮决定打造一个《经典 Bug 和排查方案汇总》。

# 大纲

1、解决 Bug 的流程套路：做项目前必读

2、经典 Bug 解决方案：根据需要查找和阅读

3、Bug 案例分享：感兴趣的同学阅读长见识

# 2、经典 Bug 解决方案

# 通用

# 项目启动时报错、无法正常运行？

Bug 解决 | 项目启动失败

# 和教程的操作完全一致，但是却出现错误？

Bug 解决 | 项目启动失败

# HTTP 响应状态码不为 200，怎么处理？

常见请求错误码解释及解决方案

# 项目本地正常运行，但上线后出现报错或无法访问？

Bug 解决 | 本地项目上线后出现错误

Bug 解决 | 无法访问线上服务

# 用户无法正常登录，无法正确获取到用户信息？

Bug 解决 | 无法正常登录或获取不到用户信息

# 后端

# 后端项目无法正常启动，或者数据库、Redis 等依赖服务连接失败？

Bug解决 | 后端项目无法正常启动、运行（含数据库、redis连接失败）

# 前端

# 使用前端框架初始化项目，出现错误、无法正常运行？

Bug 解决 | 前端框架初始化错误

# 前端无法查询到数据、一直加载或和预期结果不一致？

Bug 解决 | 数据查询为空或错误

# 使用组件库报错、或者样式丢失 / 不生效？

Bug 解决｜组件库报错、或样式丢失/不生效

# 前端项目上线后，无法正确访问到页面，或出现 404 错误？

Bug 解决｜无法正确访问到页面或出现 404 错误

# 前端项目上线后，无法正确请求后端接口并得到响应？

Bug 解决 | 前端无法正确请求后端接口并得到响应？

# 前端项目能运行，但有很多错误提示和告警？

Bug 解决 | 为什么前端项目能够正确运行，但是有很多错误提示和告警？

# 前端项目无法正确安装依赖？

Bug 解决 | 前端项目无法正确安装依赖？
